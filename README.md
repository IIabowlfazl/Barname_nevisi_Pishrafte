# پروژه درس برنامه‌نویسی پیشرفته

این مخزن مربوط به درس برنامه‌نویسی پیشرفته (مدرس: دکتر اردستانی) است و شامل سه بخش اصلی می‌باشد:

## محتویات

### ۱. پروژه‌ها (Projects)
شامل ۶ پروژه تعریف‌شده درس با استفاده از مفاهیم برنامه‌نویسی شیءگرا (OOP):

- `smart_elevator_simulator.py` — شبیه‌ساز آسانسور هوشمند (۲۰ طبقه)
- `smart_traffic_light_simulator.py` — شبیه‌ساز چراغ راهنمایی هوشمند
- `water_tank_simulator.py` — شبیه‌ساز کنترل سطح مخزن آب
- `step_response_analyzer.py` — تحلیل‌گر پاسخ گام مرتبه اول و دوم
- `inverted_pendulum.py` — شبیه‌ساز تعادل آونگ وارونه
- `metro_gate_simulator.py` — شبیه‌ساز دروازه مترو

### ۲. تمرین‌ها (Exercises)
شامل تمرین‌های برنامه‌نویسی در پوشه `exercises`:

- `first.py` — تشخیص سال کبیسه
- `binary.py` — تبدیل اعداد به باینری
- `Bin.py` — نمایش باینری اعداد
- `math.py` — محاسبه ریشه دوم
- `harmonic.py` — سری هارمونیک
- `Random.py` — تولید جایگشت تصادفی
- `jangal.py` — شبیه‌ساز اکوسیستم جانوران
- `scalar_and_matrix.py` — کار با اسکالر و ماتریس (شماره ملی)
- `newton_raphson.py` — روش نیوتن–رافسون برای یافتن ریشه
- `subsets.py` — مجموع زیرمجموعه‌ها
- `prime_finder.py` — یافتن اعداد اول
- `tax.py` — محاسبه‌گر مالیات
- `quadratic.py` — معادله درجه دو
- `ruler.py` — الگوی خط‌کش (فرکتال)
- `powers.py` — توان‌های عدد ۲
- `turtle_train.py` — ترسیم چندضلعی با turtle

### ۳. گواهینامه SoloLearn
فایل تصویر گواهینامه در پوشه `certificate` قرار دارد:
- `Sololearn.jpg` — تصویر گواهینامه
- `sololearn.txt` — نشانی (لینک) گواهینامه

## نحوه اجرا

تمرین‌ها و پروژه‌ها با پایتون ۳ اجرا می‌شوند:

```bash
python projects/smart_elevator_simulator.py
python exercises/first.py
```

بعضی از فایل‌ها ورودی از کاربر می‌گیرند و بعضی به محیط گرافیکی (GUI) نیاز دارند.
